# Meihua Forest Visualizer

This is a simple web application that displays a draggable grid with colored items.

## Running the application

1.  Make sure you have Python 3 installed.
2.  python3 -m venv meihua_venv
    source meihua_venv/bin/activate
    pip install -r requirements.txt
3.  Run the web server: `python3 server.py`
4.  Open your web browser and navigate to `http://localhost:8000`.
